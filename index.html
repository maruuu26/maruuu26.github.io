About

Developer who thinks
like a designer.


I'm a Computer Science student at Rutgers who believes the best digital products live at the intersection of great engineering and thoughtful design. While most developers think in systems, I think in user journeys.

At Hack4Impact — a student org building tech for nonprofits — I develop REST APIs for a volunteer management platform. Working alongside a dedicated frontend team, I've learned how backend decisions shape the user experience: how fast data loads, what fields exist, how errors surface.

At Wayfair, I built AI automation workflows and designed the dashboard that surfaced their insights — owning the experience from data pipeline to the interface teams actually used.

I'm pursuing UX Design to formalize a practice I've been doing instinctively: designing for real people, then building it.
